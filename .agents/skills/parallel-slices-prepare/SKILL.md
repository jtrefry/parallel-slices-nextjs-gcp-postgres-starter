# Prepare a Codex goal

Run `node scripts/parallel-slices/agent-profile.mjs require-enabled codex` before
taking any other action. Stop if Codex support is not enabled.

Read `docs/parallel-slices/prepare-controller.md` completely and follow it to
generate the exact copy-paste invocation for a Codex-owned `/goal` run. Do not
start the goal yourself.
